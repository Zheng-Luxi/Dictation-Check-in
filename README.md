# 语文听写签到系统 Chinese Dictation & Check-in System
## 一、使用 Using
### 1.打开"听写签到.html".
### 2.选择"听写"或"签到".
### 3.点击"开始"来开始"听写"或"签到".

## 二、内容
### 1.文字 -> 自选
### 2.倒计时 -> 预估
> 两字词语 5s
> 四字词语 10s
> 文言字词一个字 2s-3s 思考时间 3s
> 五言 15s
> 七言 20s
